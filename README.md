# LiftRPG

A generative Print & Play workout journal fused with a branching narrative TTRPG.

**Live site:** [liftrpg.co](https://liftrpg.co)

## Apps

| App | Path | Purpose |
|-----|------|---------|
| Renderer | `/renderer/` | Upload schema v1.3 JSON, preview booklet layouts, print/export, and unlock the encrypted ending inline |

## Development

Source: [xYoBro/LiftRPG-dev](https://github.com/xYoBro/LiftRPG-dev) (private)

This repo (`xYoBro/LiftRPG`) is the public-facing deployment only. All development happens in the private dev repo where `public/` is a git submodule pointing here.
