How it Works
  1. Write a workout program in Liftosaur or plain text
  2. Feed the template JSON + your program into any LLM (Claude, ChatGPT, Gemini)
  3. Upload the generated JSON to the Renderer
  4. Print as a saddle-stitch booklet (half-letter, double-sided)
  5. Train for six weeks, pencil in hand
  6. Assemble the password. Unlock the ending.
