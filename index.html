LiftRPG

A workout journal fused with a branching narrative TTRPG. Real training. One encrypted ending. Written in pencil.

How it Works
  1. Write a workout program in Liftosaur or plain text
  2. Use the Prompt Generator to build your LLM prompt
  3. Paste the prompt into any LLM (Claude, ChatGPT, Gemini)
  4. Upload the JSON to the Renderer — print as a booklet
  5. Train the program, pencil in hand
  6. Assemble the password. Unlock the ending.
